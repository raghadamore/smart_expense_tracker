# Smart Expense Tracker 💰

A modern Flutter application for tracking personal expenses and managing financial activities efficiently.  
The app helps users monitor spending, analyze expenses through charts, and organize transactions in a clean and responsive interface.

---

## ✨ Features

- Add, edit, and delete expenses
- Categorize expenses
- Monthly and weekly statistics
- Interactive Pie Chart and Bar Chart
- Expense history management
- Local database storage using Sqflite
- Responsive UI design
- State management using BLoC / Cubit
- Clean Architecture implementation
- Error handling
- Dark and Light friendly UI

---

## 🛠️ Technologies Used

- Flutter
- Dart
- BLoC / Cubit
- Clean Architecture
- Sqflite
- FL Chart
- Dartz
- Responsive Design

---

## 📊 Charts & Analytics

The application provides visual analytics including:

- Weekly expense bar chart
- Expense category pie chart
- Financial summaries
- Dynamic statistics updates
